MacOS

I sometimes have the need to generate a spectrogram of a song and each time, I can't remember the SoX formula. So I decided to create a small script and, with a bit of research, automate the display of spectrogram directly within the iTerm application for Mac.

Requirements

- This obviously requires macOS as I don't know about terminal application for UNIX-based OS. However, I've heard that programmes like terminator were able to do bluffing things. If you want to adapt that tutorial to your Linux OS, feel free to share it.

- iTerm for Mac. If you don't know about it and own a Mac, then please take time to discover it. It's amazing and allows many things the original Terminal.app can't do.

- The imgcat script from iTerm.

- SoX for Mac

Installation

- Install SoX on your mac using brew:

$ brew install sox --with-flac --with-lame

The flags are for compatibility purpose between lame, flac ans SoX. Don't forget them even though you already have flac and lame installed on your system.

- Install imgcat in /usr/local/bin:

$ curl https://www.iterm2.com/utilities/imgcat > imgcat $ mv imgcat /usr/local/bin $ chmod +x /usr/local/bin/imgcat

- Copy and paste that bit of code in a script file. Mine is called spectro but you can call it whatever you want. It has to be placed in /usr/local/bin.

#!/bin/bash var="$(echo $1 | sed 's/ /\\ /g')" sox "$var" -n spectrogram -o /tmp/spectrogram.png imgcat /tmp/spectrogram.png rm -rf /tmp/spectrogram.png

And make it executable:

$ chmod +x /usr/local/bin/spectro

Usage

Simply:

$ spectro /path/to/your/FLAC/or/MP3
